# Python Fundamentals — 45 Day Journey

A structured Python learning repo covering core fundamentals,
OOP, file handling, and project building.

## Topics Covered
- Data types, lists, dicts, sets, tuples
- Functions, decorators, generators
- OOP, inheritance, polymorphism
- File I/O and exception handling
- Modules and package structure

## Projects Built
- **Student Management System** — OOP + File I/O + exceptions
- **Bank Account System** — inheritance, custom exceptions, decorators
- **Word Analyzer** — strings, loops, NLP-style frequency counting

## How to Run
cd python_fundamentals
python day05.py
